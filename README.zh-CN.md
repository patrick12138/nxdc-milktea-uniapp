# NXDC Milktea UniApp

[English](./README.md) | [简体中文](./README.zh-CN.md)

一个基于 UniApp 的奈雪小程序仿制演示项目。

> **特别声明**：本项目中的展示图片归奈雪所有。

## 项目简介

本项目 fork 自 [nxdc-milktea](https://gitee.com/caoyang/nxdc-milktea)，感谢原作者。

此 fork 版本新增能力包括：

- 城市选择
- 退出登录
- 商家选择
- 微信登录 API

## 包含页面 / 模块

- 首页
- 点餐（自取 + 外卖，包含基础点餐逻辑）
- 取餐
- 我的
- 积分商城
- 积分商城详情页
- 积分签到
- 会员码
- 我的卡券
- 收货地址
- 我的资料
- 我的订单
- 订单详情
- 余额
- 会员券包
- 券包详情
- 订单评价
- 买茶送包
- 更多服务
- 开发票
- 礼品卡

## 数据说明

本仓库中的数据均为静态 mock 数据，存放在 `api/` 目录下。
