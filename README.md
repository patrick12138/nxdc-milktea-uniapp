# NXDC Milktea UniApp

[English](./README.md) | [简体中文](./README.zh-CN.md)

A UniApp-based demo project that mimics the Nayuki mini-program experience.

> **Disclaimer**: All showcase images in this project belong to Nayuki.

## Overview

This project was forked from [nxdc-milktea](https://gitee.com/caoyang/nxdc-milktea). Thanks to the original author.

Additional features in this fork include:

- City selection
- Logout
- Merchant selection
- WeChat login API integration

## Included Pages / Modules

- Order (pickup + delivery, with basic ordering flow)
